## 🚀 Installation

- Open the Extensions sidebar in VS Code
- Search for `Dionis`
- Click Install
- Open the Command Palette with Ctrl+Shift+P or ⇧⌘P
- Select Preferences: Color Theme and choose `Dionis Theme`.
